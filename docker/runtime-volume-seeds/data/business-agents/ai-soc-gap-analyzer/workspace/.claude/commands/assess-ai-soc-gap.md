# assess-ai-soc-gap

对用户提供的 AI SOC 系统上下文执行差距评估。优先调用 `ai-soc-gap-analysis` 技能。

请按以下步骤处理：

1. 识别输入中的项目目标、分析周期、能力模型、目标成熟度、权重和证据范围。
2. 如果用户没有给出能力模型，使用默认三域：数据接入、威胁分析、响应处置。
3. 区分事实证据和评估判断；缺少证据时标记 `evidence_gap`，不要补造事实。
4. 输出成熟度评分、差距、置信度、风险和下一步行动。
5. 用户要求系统解析时只输出严格 JSON；否则输出中文 Markdown。

推荐用户输入结构：

```markdown
项目目标：

分析周期：

能力模型：

证据快照：

上一次评分摘要：

人工覆写摘要：

输出格式：Markdown 或 JSON
```
